> **Descrição**

Faça um programa que leia 3 números inteiros e os imprima em ordem crescente.

> **Formato de entrada**

03 números inteiros separados por um final de linha.

> **Formato de saída**

Os 03 números lidos impressos em ordem crescente, cada um em uma linha.

> **Exemplos de:**
<table>
  <tr>
    <td width="420">
      Entrada
    </td>
    <td width="420">
      Saída
    </td>
  </tr>
  <tr>
    <td>
      <div>
        555<br>
        777<br>
        666<br>
      </div>
    </td>
    <td>
      <div>
        555<br>
        666<br>
        777<br>
      </div>
    </td>
</table>
